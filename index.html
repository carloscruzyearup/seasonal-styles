Outfitter for All Seasons!

This website demonstrates how to use jQuery to create a dynamic page that changes colors and pictures according to the different seasons. Elements are accessed by class, unique IDs and the type of element. JQuery makes this all possible without having to reinvent the wheel, a testament to the power of using frameworks to accelerate development.
